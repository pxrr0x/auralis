### Description:
***auralis is a command-line Python utility for downloading songs and playlists on Windows, Linux, and Termux.***

### Installation
#### Install dependencies (git, python):
 - For Linux
    - ```sudo apt update```
    - ```sudo apt install git python3 -y```
 - For Android(Termux)
    - <a href="https://github.com/termux/termux-app/releases">Download and install Termux here</a>
    - <a href="https://github.com/termux/termux-api/releases/">Download and install Termux:API here</a>
    - ```apt update && apt upgrade -y```
    - ```apt install git python3 -y```
  
#### Clone this repository
 - ```git clone https://github.com/pxrr0x/auralis.git```

#### Enter the directory
 - ```cd auralis```

#### Install additional dependencies and modules
 - ```python3 setup.py```

### Usage
```usage: python3 auralis.py```

1. Run the script :>
